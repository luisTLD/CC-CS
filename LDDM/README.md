# LDDM (Laboratório de Desenvolvimento para Dispositivos Móveis)

### Ambientes e Ferramentas de Desenvolvimento para Dispositivos Móveis
Estudo das principais plataformas e ferramentas utilizadas no desenvolvimento mobile, incluindo:
- Android Studio
- SDKs mobile
- Emuladores e dispositivos físicos
- Controle de versão com Git e GitHub
- Estrutura de projetos mobile

### Projeto de Interfaces
Conceitos de design e construção de interfaces para aplicações móveis:
- UX (User Experience)
- UI (User Interface)
- Componentes visuais
- Navegação entre telas
- Responsividade e usabilidade

### Desenvolvimento de Aplicações em Plataformas para Dispositivos Móveis
Desenvolvimento prático de aplicações mobile:
- Programação para Android
- Manipulação de atividades e fragments
- Integração com APIs
- Armazenamento de dados
- Gerenciamento de estados e eventos

### Introdução à Computação Ubíqua
Conceitos relacionados à computação presente no cotidiano:
- Internet das Coisas (IoT)
- Sensores e dispositivos inteligentes
- Context-aware computing
- Integração entre dispositivos
- Mobilidade e conectividade
